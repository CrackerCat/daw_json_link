176  /***

178  * String - Parser always expects a JSON string. Will surround serialized
179  * value with double quotes
180  * Literal - Parser will expect a valid JSON literal number, bool, null
181  * Any - Experimental - Parser will return any valid JSON value excluding

184  * suitable
